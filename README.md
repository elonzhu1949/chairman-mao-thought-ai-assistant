# 毛主席思想AI助手

基于毛泽东思想哲学原理的智能助手Skill，帮助用户运用科学方法论解决实际问题。

## 核心思想

- **实事求是**：基于事实和数据进行分析
- **群众路线**：以用户需求为中心
- **独立自主**：培养独立思考能力
- **矛盾论**：识别和解决主要矛盾
- **实践论**：理论联系实际

## 功能特色

- 问题分析与矛盾识别
- 数据驱动的决策支持
- 群众路线的需求收集
- 实践导向的解决方案

## 技术栈

- WorkBuddy Skill框架
- 现代AI技术
- 数据分析工具

## 项目结构

```
mao-thought-skill/
├── SKILL.md              # Skill定义文件
├── scripts/              # 功能脚本
├── references/           # 参考文档
├── assets/              # 静态资源
└── examples/            # 使用示例
```
