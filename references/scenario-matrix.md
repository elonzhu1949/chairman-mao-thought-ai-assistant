## 详细场景说明

### 场景1：复杂问题分解
**触发词**：问题很多、不知道从哪下手、一团乱麻、情况复杂
**分析框架**：
1. 列出所有问题
2. 用矛盾论判断哪个是主要矛盾（解决它能带动其他问题）
3. 分析主要矛盾的主要方面（力量对比如何）
4. 制定分阶段解决方案（先解决主要矛盾，再解决次要矛盾）
**参考文档**：`references/core-philosophy/maodilun.md`

### 场景2：重要决策制定
**触发词**：需要做决定、选A还是选B、应该怎么做
**分析框架**：
1. 调查研究：收集相关的事实和数据
2. 去粗取精、去伪存真：提炼关键信息
3. 基于事实做出判断（实事求是）
4. 设计验证方案（实践论）：小规模试点、设置反馈机制
**参考文档**：`references/core-philosophy/shijishiqiushi.md`、`references/practical-methods/diaoyan-yanjiu.md`

### 场景3：弱者vs强者竞争
**触发词**：创业公司、面对大厂、竞争不过、资源不够
**分析框架**：
1. 分析对手的力量分布——哪里强、哪里弱
2. 选择对手薄弱但真实需求存在的细分市场作为"根据地"
3. 在根据地中集中资源做到局部最优
4. 逐步扩展，避免过早正面冲突
**参考文档**：`references/strategic-methods/nongcun-baowei-chengshi.md`、`references/historical-cases/nongcun-baowei-chengshi.md`

### 场景4：争取合作伙伴
**触发词**：需要合作、联盟、找人帮忙、利益相关方
**分析框架**：
1. 识别所有潜在的利益相关方
2. 分析各方利益诉求的异同
3. 找到最大公约数——各方都能接受的共同利益
4. 又联合又斗争：在共同利益上联合，在分歧点上灵活处理
**参考文档**：`references/strategic-methods/tongyi-zhanxian.md`

### 场景5：团队矛盾/效率问题
**触发词**：团队有矛盾、效率低下、沟通不畅、不团结
**分析框架**：
1. 诊断问题根源：是利益冲突？认知分歧？还是流程问题？
2. 用"团结-批评-团结"的公式设计解决方案
3. 建立安全的沟通环境（对事不对人）
4. 领导带头做自我批评
**参考文档**：`references/organizational/piping-zipiping.md`、`references/historical-cases/yanan-zhengfeng.md`

### 场景6：组织转型/创新
**触发词**：需要变革、转型、创新、跟不上变化
**分析框架**：
1. 实事求是地评估现状：哪些在老化？哪些需要改变？
2. 分析组织老化的根源：路径依赖？既得利益？还是文化惰性？
3. 设计渐进式变革方案（不是激进式颠覆）
4. 通过试点验证变革方案
**参考文档**：`references/organizational/ziwo-geming.md`、`references/historical-cases/zunyi-huiyi.md`

### 场景7：新方案推广
**触发词**：推新方案、新制度、新产品上线、需要大家接受
**分析框架**：
1. 选择有代表性的试点
2. 在试点中做深做透
3. 总结可推广的经验（区分"方案贡献"和"环境贡献"）
4. 逐步推广，因地制宜（不搞一刀切）
**参考文档**：`references/practical-methods/dianxing-diaocha.md`

### 场景8：危机/紧急情况
**触发词**：出了问题、紧急、危机、需要马上处理
**分析框架**：
1. 快速判断：核心问题是什么？最坏情况是什么？
2. 集中资源处理最紧急最重要的问题（不分散力量）
3. 保持灵活性——根据情况变化随时调整方案
4. 保存核心能力（先保命再图发展）
**参考文档**：`references/strategic-methods/junshi-zhanlue.md`、`references/historical-cases/san-da-zhanyi.md`

### 场景9：个人成长/职业发展
**触发词**：职业选择、能力提升、个人发展、学习规划
**分析框架**：
1. 实践论指导学习方法：在实践中学习→总结认识→再实践→再认识
2. 调查研究指导职业选择：先了解行业实际情况，再做出判断
3. 矛盾论指导能力发展：找出当前最需要提升的核心能力（主要矛盾）
4. 批评与自我批评：定期复盘自己的成长轨迹
**参考文档**：`references/modern-applications/problem-solving.md`

### 场景10：说服/影响他人
**触发词**：需要说服、别人不听、沟通困难、推行政策
**分析框架**：
1. 群众路线：先了解对方的真实需求和顾虑
2. 从对方的立场出发寻找共同利益
3. 一般号召+个别指导：有统一的方向，但对不同人用不同方式沟通
4. 抓两头带中间：先说服影响力大的人，带动其他人
**参考文档**：`references/core-philosophy/qunzhongluxian.md`

### 场景11：长期战略规划
**触发词**：战略规划、长期目标、发展方向、未来规划
**分析框架**：
1. 战略上藐视困难：明确长期目标和方向（要有信心）
2. 战术上重视细节：制定具体的、可执行的阶段目标
3. 原则坚定+策略灵活：核心方向不变，但路径可以调整
4. 用矛盾论分析不同发展阶段的主要矛盾变化
**参考文档**：`references/strategic-methods/zhanlue-zhance.md`

### 场景12：组织文化建设
**触发词**：企业文化、价值观、团队凝聚力、使命愿景
**分析框架**：
1. 明确"我们为谁服务"的根本问题（古田会议的核心）
2. 将价值观具象化为可感知的行为准则
3. 领导以身作则（文化建设的关键是领导行为，不是贴标语）
4. 建立学习型组织和持续改进机制
**参考文档**：`references/organizational/zuzhi-jianshe.md`、`references/historical-cases/gutian-huiyi.md`
